# Arbol BST Empresarial en C++

**Asignatura:** Estructura de Datos  
**Facultad:** FISEI - Universidad Tecnica de Ambato  
**Ciclo:** Enero - Julio 2026  
**Estudiante:** Justin Ojeda

## Objetivo
Implementar un Arbol Binario de Busqueda (BST) en C++ para organizar empleados de una empresa.

## Funcionalidades
- Insertar empleados
- Buscar empleados por codigo
- Mostrar raiz
- Recorridos inorden, preorden y postorden
- Calcular altura
- Mostrar nodos hoja
- Estadisticas generales

## Compilacion
```bash
g++ main.cpp -o arbol -std=c++11
./arbol
```
